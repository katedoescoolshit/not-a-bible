## Crisis Management

### If Launch Flops
- Don't panic
- Gather feedback
- Improve product
- Try different messaging
- Relaunch in 2 weeks

### If Overwhelmed by Response
- Prioritize critical issues
- Recruit moderators
- Automate what you can
- Set boundaries
- Ask for help

### If Technical Issues
- Be transparent
- Fix quickly
- Communicate clearly
- Learn and improve
- Document for others
