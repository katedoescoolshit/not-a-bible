## Key Principles

1. **Preservation First** - Built to last decades, not years
2. **Zero Dependencies** - Static Bible has no external requirements
3. **Privacy Focused** - No tracking, no surveillance
4. **Maximum Portability** - Works anywhere, on any device
5. **Open Source** - Free to copy, share, and modify

## Tech Stack

### Static Bible Reader
- Pure HTML5
- Pure CSS3
- Vanilla JavaScript (ES6+)
- LocalStorage for bookmarks

### Next.js Landing Site
- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Vercel hosting
